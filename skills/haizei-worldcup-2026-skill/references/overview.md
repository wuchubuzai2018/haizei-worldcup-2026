# 赛事基本信息

## 基本信息

- **赛事名称**：2026年国际足联世界杯（美加墨世界杯）
- **比赛时间**：北京时间2026年6月12日至7月20日
- **主办国家**：美国（11城）、加拿大（2城）、墨西哥（3城）
- **参赛球队**：48支
- **分组数量**：12组（A-L），每组4队
- **晋级规则**：每组前2名 + 8支成绩最好的小组第3名晋级32强
